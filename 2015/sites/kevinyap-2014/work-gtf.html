Georgetown Festival

2013

George Town Festival is a month-long festival which began as a celebration of George Town’s UNESCO World Heritage Site Listing in 2008.

My role in this project is implement the theme which is provided by the client on Wordpress. Other than that, I have to do some modification on the theme itself so that the layout is same to the design they provided. On top of that, I also have to did a complete backup of previous content for the client.

Disciplines: Coding Client: Joe Sidek Productions Sdn. Bhd. Skill used: Photoshop, HTML, CSS, PHP
